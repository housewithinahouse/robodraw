Color Test | New Expeirments


— 2012 —

These are the first experiments we conducted using color. We found that gel pens yield very clean results and layer nicely. Sharpies are great for drawing thick lines.

See our color spirals gallery for more finished color work.

This page is mostly a dumping ground for color tests.

These were designed using Inkscape + Python,
and executed by a modified Mimaki CG-45 Vinyl Cutter

This set of spiral forms was drawn with whatever junk I found
and Moonglow and Metallic Sakura Gelly-Roll Pens
on 80lb Sawgrass Cardstock.
