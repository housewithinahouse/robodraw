Don't Die | Linework
— 2012 —

This project focuses on the warping and intersection of lines. Interruptions in a line's flow feel jarring. The photo below shows the beginning stages of making a drawing like this, when the guide-text is still on the page.

"Don't Die" is a phrase I use whenever I need text for something. It seems like good advice.

This is still an ongoing series; come back later for more prints.

These were designed using Inkscape + Python,
and executed by a modified Mimaki CG-45 Vinyl Cutter

These lines and words were drawn using a Uniball Deluxe Fine
on 80lb Sawgrass Cardstock.
