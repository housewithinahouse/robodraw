Tags

< pre > ::: "preformatted text", usually displayed in a monospace font,
	      preserves whitespace and line breaks.

< code > ::: "code", monospace. doesn't worry about whitespace.

< html > ::: can have a "id"

< head > ::: should contain a < title > element. can also take < base />, < link />, < style />, < script />, and < meta />.

< title > ::: it is at the top of the window, the default name for bookmark, used to help index pages.

< link > ::: used to include other code in your document. has < rel >, the relation; and < href > usually.

< script > ::: < script src="js/main.js">< /script>

Attributes

There are 4 core attributes that almost every HTML element has:

  • id
    • id is used to uniquely ID an element within a page
    • id="webapp"
    • must begin with a letter. can contain numbers, hyphens, underscores, colons or periods in addition to letters. must be unique to that document.
  • class
    • class is used to specify that an element belings to a group of elements
    • if we had a bunch of paragraphs that we wanted to seperate out, we could use a class.
      < p class="summary" > words < /p>
      < p> text < /p>
      < p class="summary" > more words < /p>
    • space seperated list of words can be used for value: class="className1 className2"
  • title
    • title is used for the suggested name of item, useful if it doesn't load
  • style
    • allows you to specifiy CSS rules in the document
    • < p style="font-family:arial;">text< /p>
